## 简单定律

### 同一律
输入
```
p->p
```
输出
```
(  1°)  p->(#327->p)    [l1]
(  2°)  p->((#327->p)->p)       [l1]
(  3°)  (p->((#327->p)->p))->((p->(#327->p))->(p->p))   [l2]
(  4°)  (p->(#327->p))->(p->p)  [MP 2°, 3°]
(  5°)  p->p    [MP 1°, 4°]
```

### 否定前件律
输入
```
!q->(q->p)
```
输出
```
(  1°)  !q->(!p->!q)    [l1]
(  2°)  (!p->!q)->(q->p)        [l3]
(  3°)  ((!p->!q)->(q->p))->(!q->((!p->!q)->(q->p)))    [l1]
(  4°)  !q->((!p->!q)->(q->p))  [MP 2°, 3°]
(  5°)  (!q->((!p->!q)->(q->p)))->((!q->(!p->!q))->(!q->(q->p)))        [l2]
(  6°)  (!q->(!p->!q))->(!q->(q->p))    [MP 4°, 5°]
(  7°)  !q->(q->p)      [MP 1°, 6°]
```
